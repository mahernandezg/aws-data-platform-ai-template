# Terraform Templates

## Purpose

This directory contains Terraform starters aligned to the repository architecture.

## Current Starters

- reusable IAM role-only module in `modules/dcs_iam_role`
- near-runnable IAM role example in `examples/dcs_glue_runtime_role`

## Standards

- IAM Roles only
- explicit naming and tagging patterns
- environment-aware structure
- secure-by-default examples where feasible

## Intended Use

Use these templates as a starting point for:

- runtime roles
- role-based access patterns
- architecture-aligned infrastructure scaffolding

Do not treat these templates as a replacement for environment-specific enterprise modules.
