# Glue Templates

## Purpose

This directory contains AWS Glue job starters aligned to the repository architecture.

## Current Starter

- `isc_iceberg_ingestion_job.py`

## Standards

- architecture component declared in the file
- environment applicability declared in the file
- IAM Roles only
- Apache Iceberg-first output logic
- explicit source parsing placeholder
- lineage and observability extension points

## Intended Use

Use these starters to accelerate Glue jobs that fit the repository architecture without teaching unsafe or architecture-blurring patterns.
